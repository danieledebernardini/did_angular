This site allows user to deploy, verify and delete decentralized identifiers on the XRPL devnet.

DID Handler

Any owner of an XRPL account can issue their decentralized identifier.

This is just a demo: the generation of a decentralized identifier should be done locally (e.g. within a wallet), since it requires the secret seed of the user's account.

DID Verifier

Anyone can check whether a decentralized identifier matches with some identifying information.

After creating your DID you can test them from here.
